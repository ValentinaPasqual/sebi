Introduction back to ToC

The Scholarly Evidence Based Assessment (SEBI) Ontology addresses the features characterising the interpretative activities performed by scholars in identifying the authenticity of a document. SEBI, as an extension of HiCo ontology, allows the representation and reasoning about the reliability of evidences which support scholars' interpretations. In particular SEBI in provided with a set of controlled vocabularies which shape evidences as the relation of (documentary) features (e.g. style, content) with a set of attributes (e.g. consistency, veridicality). Additionally, SEBI allows to represent negation via an evaluation score (e.g. the evidence (sebi:Evidence) tackles the realiability (sebi:Attribute) of the document's contents (sebi:Feature) with score 0, meaning that the contents of the document are unreliable).

SEBI: Description back to ToC

The Scholarly Evidence Based Interpretation (SEBI) ontology is an OWL 2 DL ontology developed to represent evidences used to support scholarly interpretations.
